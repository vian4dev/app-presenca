## Instalação
Clone o repositório
~~~
git clone https://github.com/vian4dev/react-app-presenca.git
~~~
Acesse o diretório do projeto
~~~
cd react-app-presenca
~~~
Instale as dependências
~~~
npm install
~~~
Execute a aplicação
~~~
npm run dev
~~~
