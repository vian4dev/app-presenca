## Instalação
Clone o repositório.
~~~
git clone https://github.com/vian4dev/app-presenca.git
~~~
Acesse o diretório do projeto.
~~~
cd app-presenca
~~~
Instale as dependências.
~~~
npm install
~~~
Execute a aplicação.
~~~
npm run dev
~~~
